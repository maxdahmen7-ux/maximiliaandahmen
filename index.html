How the automation works 6 stages · end-to-end
01↓
Input · The Eyes
Student uploads a photo
A photo of homework or a textbook page arrives via the Web Library or WhatsApp. No manual re-entry — the image is the trigger.
WhatsApp API Web Library
02↓
Analysis · Vision & OCR
The AI reads the image
As the image hits the Replit server, a vision model processes the pixels: text is extracted via OCR and the subject and level are identified automatically.
GPT-4o Vision Gemini Pro Vision OCR
03↓
Transformation Layer
Four engines run in parallel
The analyzed content fans out to four specialised engines simultaneously.
Smart Summarizer
Filters key points into a concise, structured summary.
Quiz Generator
Creates multiple-choice questions from the source text.
Flashcard Engine
Converts concepts into flashcards optimised for long-term retention.
AI Buddy
Loads the content as context for the Socratic chatbot.
04↓
Gamification · The Hook
Survival Mode keeps students engaged
Content is delivered inside a game loop. Wrong answers cost lives; correct streaks build momentum. The format makes practice feel earned, not forced.
❤ Lives 🔥 Streaks Survival Mode
05↓
Retention · Socratic Method
The AI Buddy guides, never just tells
When a student gets stuck, the AI Buddy intervenes. Because it already knows the photo context, it asks targeted questions that lead the student to the answer themselves — no spoon-feeding.
06
Distribution · Accessible Everywhere
The full chain lives in your pocket
A message to the WhatsApp Business number triggers the entire pipeline within milliseconds. No app to download, no account to create.
WhatsApp API < 500ms response Zero friction
